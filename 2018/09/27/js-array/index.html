常用方法

filter

filter方法会创建一个新的数组，新数组中的元素是通过检测指定数组中的所有元素获取的。
注意:

  1. 不会对空数组进行检测
  2. 不会改变原始数组
    code:

map

map是定义一个新的数组，新数组的元素是通过调用函数处理原数组得到的值
注意:

  1. 不会对空数组进行检测
  2. 不会修改原始数组
  3. 新数组的长度等于原始数组的长度
    code:

some

some 会检测数组中的元素是否满足条件(函数提供),会依次遍历数组的每个元素，如果，有一个元素满足条件则返回true,没有元素满足条件则返回false.
code:

every

every用来检测数组中的所有元素是否满足指定条件(函数提供),如果有一个元素不满足条件则返回false,都满足则返回true.
