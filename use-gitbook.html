如何使用GitBook

  1. 安装node
  2. 安装gitbook-cli:npm install gitbook-cli -g
  3. 查看是否安装成功:gitbook -V
  4. 创建:gitbook init
  5. 本地浏览:gitbook serve
  6. 构建:gitbook build
